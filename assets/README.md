# Assets do 5task

Esta pasta contém todos os recursos visuais necessários para o funcionamento offline do aplicativo e sua posterior conversão para plataformas nativas (Android/iOS).

## Arquivos Necessários:
1.  `Stalk logo.png` - Logo principal do app.
2.  `einstein-happy.png` - Avatar para estado de sucesso.
3.  `einstein-skeptical.png` - Avatar para estado inicial/pensativo.
4.  `einstein-ecstatic.png` - Avatar para estado de entusiasmo.
5.  `einstein-worried.png` - Avatar para estado de limite de tarefas.

> **Nota:** Estes arquivos devem ser mantidos aqui para que o Service Worker consiga cachear o app corretamente para uso sem internet.
